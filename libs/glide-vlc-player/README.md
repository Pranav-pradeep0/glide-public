# @glide/vlc-player

First-party Glide VLC playback module used by the Glide app.

## Install (local workspace)

```bash
npm install
```

The app consumes this package via:

```json
"@glide/vlc-player": "file:./libs/glide-vlc-player"
```

## Exports

- `VLCPlayer`
- `VlCPlayerView`
- Types from `index.d.ts`
